## Estrutura

- `TPC1/` - website estático gerado a partir do dataset de reparações automóveis com Python
- `TPC2/` - servidor aplicacional Node.js com `http` e `axios` para exploração do dataset de reparações
- `TPC3/` - servidor aplicacional Node.js para a Escola de Música, consumindo dados servidos por `json-server`
- `TPC4/` - aplicação web para gestão de exames médico-desportivos com Node.js, `json-server` e Pug
- `TPC5/` - aplicação Express com Pug para exploração de filmes, atores e géneros
