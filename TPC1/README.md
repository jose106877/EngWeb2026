## Resumo

Este trabalho consiste na análise do dataset `dataset_reparacoes.json` e na criação de um conjunto de páginas HTML estáticas para consulta da informação sobre reparações automóveis, clientes, viaturas e tipos de intervenção.

O processamento dos dados é feito com o script `json2html.py`, que lê o ficheiro JSON e gera automaticamente um website em `output/`. A solução atual já produz uma página principal, listagens para consulta e páginas de detalhe geradas a partir dos registos existentes no dataset.

## Lista de Resultados

- `document.pdf` — enunciado do TPC1
- `dataset_reparacoes.json` — dataset base das reparações
- `json2html.py` — script Python responsável pela geração do website

- `output/listaIntervencao.html` — listagem dos tipos de intervenção
- `output/` — conjunto de páginas HTML geradas automaticamente
